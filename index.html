AR.js + A-Frame — Starter Kit

Three tiny demos you can open on your phone (HTTPS required for camera):

  1. Marker-based (HIRO marker)
  2. Image tracking (T-Rex)
  3. Location-based (GPS)

Tip: If you open these files locally, use a simple local server (not just double-click). For example, in VS Code use “Live Server”, or try python -m http.server in this folder.
